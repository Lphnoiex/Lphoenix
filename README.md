# Lphoenix
My first GitHub project,a framework for univariate time series forecasting.A lightweight model framework based on GRU and lightweight single-head self-attention(using temperature forecasting in Baoshan District,Shanghai as an example).
My name is Wang Yanbo,from Shanghai University.Welcome to modify and imporve this project!
In fact,for temperature forecasting,it can be modified to multivariate for lag coupling,which yields better results.However,this is just a framework.
